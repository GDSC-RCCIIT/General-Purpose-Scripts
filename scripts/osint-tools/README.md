## Setup 🛠️
- Install decencies by running 
    ```
    pip install -r requirements.txt
    ```
- Navigate to `General-Purpose-Scripts/scripts/OSINT_TOOLS`

- Run 
    ```
    python3 main.py
    ```
and you are good to go
